# OrangeHRM Automation Testing Project

## Project Overview
This project contains automated test scripts for the OrangeHRM demo website using **Selenium WebDriver** and **pytest**.  
The purpose of this project is to validate the login functionality and basic navigation features of the application.

## Application Under Test
- URL: https://opensource-demo.orangehrmlive.com
- Browser: Google Chrome
- Test Framework: pytest
- Automation Tool: Selenium WebDriver
- Language: Python
